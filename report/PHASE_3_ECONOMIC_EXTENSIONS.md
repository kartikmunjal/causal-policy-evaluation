# Phase 3 Economic Extensions

Phase 3 adds the economic contribution layer after the Phase 1 baseline and Phase 2 infrastructure.

It asks whether minimum-wage effects are more visible through mechanisms and heterogeneity than through the average employment coefficient alone.

Phase 3 outputs:

- `phase2_margin_decomposition.csv`
- `phase2_bite_dose_response.csv`
- `phase2_heterogeneity.csv`
- `phase2_spillover_identification.csv`
- `phase2_specification_curve.csv`
- `phase2_specification_curve.png`

The filenames retain the `phase2_` prefix from the script version that first generated them, but conceptually this README treats them as Phase 3 research extensions.

Economic questions:

- Do higher-bite minimum-wage increases produce larger responses?
- Do effects show up in establishments, employment per establishment, or average pay?
- Do larger-establishment counties adjust differently?
- Do stronger pre-period growth counties absorb minimum-wage changes differently?
- Is a border-spillover test identified in the available sample?
- Are conclusions stable across sample windows and weighting choices?
